## [+] Installation

 - For KALI Linux :
    - ```git clone https://github.com/AbdulazizDXT/KaliOnTermux```
    - ```cd KaliOnTermux```
    - ```chmod +x installer.sh```
    - ```./installer.sh```

#### Or, directly run :
```
git clone https://github.com/AbdulazizDXT/KaliOnTermux ;cd KaliOnTermux ;chmod +x installer.sh ;./installer.sh

```
